# Knowledge Showcase

Public HTML showcase pages exported from selected `knowledge-mirror` notes.

Only share-safe HTML artifacts are published here. Private notes, attachments, credentials, and raw workspace files are intentionally excluded.

## How to view

請用 GitHub Pages 連結開啟，這樣會直接是網頁：

- Site: https://chenhom.github.io/knowledge-showcase/
- Voice workflow page: https://chenhom.github.io/knowledge-showcase/ai/voice-workflow-design.html

不要用 `github.com/.../blob/.../*.html` 當分享連結；那是 GitHub 原始碼檢視頁，不是網頁成品。
